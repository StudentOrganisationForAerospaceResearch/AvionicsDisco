# AvionicsDev / AvionicsDisco / AvionicsDiscovery

Discovery board support for AvionicsSoftware

## Setup
1. Initialize the AvionicsSoftware submodule by using the command
```
git submodule init
```
2. Change huart5 to huart6 in main_avionics.hpp

## Relevant Pinout
DEBUG_UART_RX = PC6
DEBUG_UART_TX = PC7
